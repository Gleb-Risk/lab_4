Конструкторы
	
Time()	Конструктор по умолчанию: устанавливает 00:00.
Time (short h, short m)	Инициализирует время часами и минутами 
Time (int totalMinutes)	Инициализирует из общего числа минут к диапазону[0, 1439]

 Публичные методы
void print() const	Выводит время в формате ЧЧ:ММ 

Унарные операторы 
Time& operator++() добавляет 1 минуту
Time operator++(int) сохраняет копию, увеличивает текущий объект, возвращает старую копию
Time& operator--() вычитает 1 минуту; при 00:00→23:59
Time operator--(int) сохраняет копию, уменьшает текущий объект, возвращает старую копию

 Операторы приведения типов
operator int() const	Неявное приведение к int: возвращает общее число минут с начала суток 
explicit operator bool() const	Явное приведение к bool: возвращает true, если время не 00:00 и false – если 00:00 
 Бинарные операторы сравнения с int
bool operator<(int minutes) const	Сравнивает общее число минут данного времени сminutes
bool operator > (int minutes) const	То же, но для >
 Бинарный оператор вычитания (-)
	
Time operator-(const Time& other) const	Вычисляет разность:(*this) - other, в минутах

Вспомогательные функции
void normalize()	Приводит hours_иminutes_  к корректному диапазону:
1.	Переводит всё в общее число минут
2.	Берёт % 1440
3.	Если результат < 0 — прибавляет 1440
4.	Вычисляет новые hours_, minutes_
Гарантирует: 0 ≤ hours_ ≤ 23,  0 ≤ minutes_ ≤ 59

void ClearInput()  Очищает буфер ввода std::cin после ошибки
short InputHours() Получение от пользователя корректного значения часов
1.	Запускает бесконечный цикл while (true).
2.	Если пользователь ввёл не число, операция провалится, cin перейдёт в состояние ошибки, и условие будет false
short InputMinutes Получение от пользователя корректного значения минут
1.	Запускает бесконечный цикл while (true).
2.	Если пользователь ввёл не число, операция провалится, cin перейдёт в состояние ошибки, и условие будет false
